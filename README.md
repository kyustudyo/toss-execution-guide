# Toss Execution Guide

Create Toss-style single-file HTML execution guides that help readers immediately understand what to do next.

## Included

- Codex skill
- Claude slash command
- Codex / Claude / Gemini prompts
- Installation guide

## Goal

Turn any product, feature, SDK, architecture, or implementation topic into a highly readable HTML guide with:

- one-line summary
- scope and non-scope
- AS-IS / TO-BE
- execution plan
- diagrams
- implementation / QA / ops checklists
- risks and open decisions
